Guidelines for input:
1. formula to be checked must be written in one line.
2. All the variable names in the formula must start with 'x'
3. All the variables must be written within '(' and ')'
4. Operator precedence is assumed by the parenthesization.
5. Input file name should be "test.txt"
6. Boolean "and","or" must be written as AND and OR respectively.
====================================
Some of the test formulae are given in "all_tests.txt" file.
One can copy and paste any of the formula to test.txt and run tautology_check.cpp.
Result will be shown in the cmd.
